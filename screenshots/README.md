# Screenshots Folder

This folder contains all the required screenshots for the VyomGarud landing page.

## Required Screenshots:
1. hero.png - Hero section screenshot
2. about.png - About section screenshot
3. capabilities.png - Capabilities/cards section screenshot
4. highlights.png - Highlights section screenshot
5. footer.png - Footer/Contact screenshot
6. mobile.png - Mobile responsive view (optional bonus)

## How to Capture Screenshots:

1. Run the development server: `npm run dev`
2. Open http://localhost:3000 in your browser
3. Take full-width screenshots of each section
4. Save them in this folder with the names listed above
5. Recommended resolution: 1920x1080 for desktop views
6. For mobile: Use browser dev tools (F12) and set to mobile view (375x667 or similar)

## Tips:
- Use browser's full-page screenshot feature for best results
- Ensure all animations have loaded before capturing
- Capture in a clean browser window (no extensions visible)
- Make sure the orange accent color (#ff7b00) is clearly visible
